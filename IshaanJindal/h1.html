Computer Languages

  1. Machine Language (Low-Level Language)
    • The most basic form of code, directly understood by a computer.
    • Written in binary (0s and 1s).
    • No abstraction, hardware-specific.
  2. Assembly Language
    • Uses mnemonics (abbreviations) instead of binary.
    • Still low-level but easier than machine code.
    • Requires an assembler to convert to machine code.
  3. High-Level Programming Languages
    • Closer to human language, easier to learn and use.
    • Requires a compiler or interpreter to execute.
